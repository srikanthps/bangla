# Appendix A – Bengali Script (Optional)

# Welcome

Congratulations on completing the spoken Bangla course!

Everything in the main course was designed to help you **understand and speak Bangla** without first learning a new writing system.

This appendix is completely optional.

If your goal is simply to converse with Bengali speakers, enjoy Bengali movies, and understand Bengali songs, you can safely skip this appendix.

However, if you'd like to:

- Read Bengali book titles
- Read movie posters
- Read song lyrics
- Read shop signs
- Read newspapers and books

then learning the Bengali script is the next logical step.

The good news is that the Bengali script is highly systematic.

---

# The Bengali Script at a Glance

The Bengali writing system is an **abugida**.

This means:

Every consonant automatically carries the vowel sound **অ (ô)** unless another vowel sign is added.

For example:

```
ক
```

is not just

```
K
```

It is pronounced approximately as

```
Kô
```

To change the vowel, a vowel sign is attached.

For example:

```
কা
কি
কী
কু
কূ
কে
কো
```

All begin with the consonant **ক (k)** but have different vowel sounds.

# Independent Vowels

These are used when a word begins with a vowel.

| Letter | Roman | Example |
|---------|--------|----------|
| অ | ô | ônek |
| আ | a | aam |
| ই | i | ilish |
| ঈ | ī | ঈশ্বর (Ishwar) |
| উ | u | utsho |
| ঊ | ū |  |
| ঋ | ri | rishi |
| এ | e | ek |
| ঐ | oi | oishwarya |
| ও | o | ora |
| ঔ | ou | oushodh |

Don't worry about memorizing them all immediately.

---

# Vowel Signs (Kar)

When a vowel follows a consonant, a vowel sign is attached.

Using **ক (k)** as an example:

| Letter | Roman | Sound |
|---------|--------|-------|
| ক | kô | kô |
| কা | ka | kaa |
| কি | ki | ki |
| কী | kī | kee |
| কু | ku | ku |
| কূ | kū | koo |
| কে | ke | kay |
| কৈ | koi | koy |
| কো | ko | ko |
| কৌ | kou | kow |

This pattern works for nearly every consonant.

# Common Conjunct Letters

As you continue reading, you'll encounter combined consonants called **conjuncts**.

Some common examples include:

| Bengali | Roman |
|----------|--------|
| ক্ত | kt |
| ন্দ | nd |
| ন্ত | nt |
| স্ত | st |
| দ্ধ | ddh |
| ক্ষ | kkh / khy (depends on the word) |

Don't worry about mastering all conjuncts at once.

Learn them gradually through reading.

# Reading Practice

Try reading these without looking at the Roman transliteration first.

```
আমি বাংলা শিখছি।
```

Roman:

Ami Bangla shikchhi.

English:

I am learning Bangla.

---

```
তুমি কেমন আছো?
```

Roman:

Tumi kemon acho?

English:

How are you?

---

```
ধন্যবাদ।
```

Roman:

Dhonnobad.

English:

Thank you.

---

```

```

Roman:

Cholo jai.

English:

Let's go.

---

```
আমি চা চাই।
```

Roman:

Ami cha chai.

English:

I want tea.

# Practice Every Day

Spend just five minutes each day reading Bengali.

Start with:

- shop signs
- advertisements
- movie posters
- YouTube thumbnails
- song titles

Because you already know the spoken language, reading will become easier much faster than it would for a complete beginner.

# Final Thoughts

Learning the Bengali script is **not required** to enjoy spoken Bangla.

However, once you can read even simple words, an entirely new world opens up.

You'll be able to:

- Read movie titles.
- Follow song lyrics.
- Read menus.
- Understand signboards.
- Explore Bengali books and newspapers.
- Appreciate the language more deeply.

Take your time.

Read a little every day.

Within a few weeks, you'll be surprised by how much Bengali text you can recognize.
