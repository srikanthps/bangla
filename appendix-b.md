# Appendix B – Roman Pronunciation Reference

# Welcome

Throughout this course, Bangla has been written using **Roman transliteration** (the English alphabet).

This made it possible to begin speaking immediately without first learning the Bengali script.

This appendix serves as a complete pronunciation reference whenever you're unsure how a word should sound.

Remember:

Roman letters are only an approximation.

The best way to improve pronunciation is by listening to native speakers.

---

# How to Use This Guide

Whenever you see a Bangla word written in Roman letters:

1. Read it slowly.
2. Pronounce each syllable clearly.
3. Avoid adding English-style stress.
4. Listen to native pronunciation whenever possible.
5. Repeat until the rhythm feels natural.

---

# The Most Important Rule

Unlike English,

Bangla pronunciation is usually consistent.

For example,

English:

- though
- through
- rough
- cough

The spelling doesn't reliably tell you the pronunciation.

Bangla is much more predictable.

Once you learn the sounds, you'll be able to pronounce most words correctly.

# Aspirated Consonants

One feature of Bangla is aspiration.

Compare:

| Roman | Approximate Sound |
|--------|-------------------|
| k | k |
| kh | k + breath |
| g | g |
| gh | g + breath |
| t | t |
| th | t + breath |
| d | d |
| dh | d + breath |
| p | p |
| ph | p + breath |
| b | b |
| bh | b + breath |

These are separate sounds.

Changing aspiration can change the meaning of a word.

---

# Word Stress

English stresses one syllable strongly.

Bangla usually doesn't.

English:

TA-ble

Bangla:

bha-lo

Both syllables receive almost equal emphasis.

This gives Bangla its smooth rhythm.

---

# Pronounce Every Syllable

English often swallows sounds.

Bangla usually doesn't.

Instead of saying

```
Am
```

say

```
A-mi
```

Instead of

```
Tum
```

say

```
Tu-mi
```

Clear syllables improve pronunciation dramatically.

---

# Speaking Rhythm

Imagine Bangla as flowing water.

Avoid speaking in short, choppy bursts.

Instead of

```
AMI...

BANGLA...

BOLI...
```

say

```
Ami Bangla boli.
```

as one smooth phrase.

---

# Words Beginners Often Mispronounce

| Roman | Correct Idea |
|--------|--------------|
| dhonnobad | Don't split "dh" into two sounds |
| bhalobasha | "bh" is one sound |
| shundor | Both syllables are soft |
| Kolkata | Don't over-stress "Kol" |
| brishti | Keep the "sh" soft |
| swapno | The "pn" is spoken together |
| hridoy | Light "h", light "r" |

---

# Silent Letters?

Unlike English,

Bangla has very few truly silent letters.

Most letters are pronounced.

If you're unsure,

pronounce every syllable clearly.

You'll usually be understood.

```

```

---

```

---

```

*The little boy is drinking tea.*

Don't worry about speed.

# Final Advice

Don't aim for a perfect accent.

Aim to be **clear**.

Native speakers are used to hearing different accents.

If you pronounce words clearly, speak at a comfortable speed, and use natural sentence patterns, you'll be understood.

Good pronunciation comes from thousands of small moments of listening and speaking—not from memorizing rules.

Keep listening.

Keep repeating.

Your pronunciation will continue improving every day.
